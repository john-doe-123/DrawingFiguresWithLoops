# Drawing figures with loops


## 1. [TenByTenRectangle](src/TenByTenRectangle.java):

### Description:

Draws a 10 x 10 rectangle.

#### Output:

```
**********
**********
**********
**********
**********
**********
**********
**********
**********

```

```

****
****
```
